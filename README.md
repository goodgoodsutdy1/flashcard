# 英语闪卡

个人英语学习工具，基于 SM-2 间隔重复算法，纯前端实现，数据存储在浏览器本地。

## 功能

- **间隔重复**：SM-2 算法自动安排复习时间，记得越好复习间隔越长
- **卡片朗读**：调用浏览器内置 Web Speech API，免费朗读英语发音
- **3D 翻转动画**：点击卡片翻转查看答案
- **预置卡组**：开箱即用，包含元音、辅音、高频词汇三个卡组
- **自定义卡组**：支持新建卡组、增删卡片
- **学习统计**：7天学习记录图表
- **PWA**：可安装到手机桌面，离线使用

## 快速开始

```bash
npm install
npm run dev
```

浏览器打开 `http://localhost:5173`，点击 **Add →** 加载预置卡组，然后点击 **Review** 开始复习。

## 复习流程

1. 点击卡片 → 翻转显示释义
2. 点击 🔊 按钮朗读单词
3. 根据记忆程度评分（0–5）：

| 分数 | 含义 | 下次复习 |
|------|------|----------|
| 0–2  | 没记住 | 明天再来 |
| 3    | 勉强记住 | 间隔略增 |
| 4–5  | 记住了 | 间隔大幅增加 |

## 预置卡组

| 卡组 | 数量 | 内容 |
|------|------|------|
| English Vowels | 10 张 | IPA 元音音标，如 /æ/ /iː/ /ə/ |
| English Consonants | 18 张 | IPA 辅音音标，如 /θ/ /ʃ/ /dʒ/ |
| Basic English Words | 50 张 | 常用高频词汇，含音标和例句 |

## 构建与部署

```bash
npm run build   # 生成 dist/ 目录
npm run preview # 本地预览生产版本
```

`dist/` 目录为纯静态文件，可直接部署到 Netlify、GitHub Pages、Vercel 等任意静态托管。

## 安装到手机桌面

部署后用手机浏览器打开，点击菜单中的 **"添加到主屏幕"**，即可像原生 App 一样使用（支持离线）。

## 技术栈

| 技术 | 用途 |
|------|------|
| React 18 + TypeScript | UI 框架 |

| Tailwind CSS | 样式，移动端优先 |
| react-router-dom | 页面路由 |
| Web Speech API | 单词朗读（浏览器内置，免费） |
| localStorage | 数据持久化 |
| vite-plugin-pwa | PWA 支持 |
